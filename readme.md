# AI Content Generator

Generate marketing content using Claude API.

## Features
- Product descriptions
- Social media posts
- Ad copy
- Blog outlines

## Tech Stack
- Python, Flask, Claude API

## Setup

1. Clone: `git clone https://github.com/YOUR-USERNAME/ai-content-generator.git`
2. Create .env: `cp .env.example .env` and add API key
3. Install: `pip install -r requirements.txt`
4. Run: `python app.py`
5. Open: http://localhost:5000

## Usage
1. Select content type
2. Enter topic/description
3. Click "Generate Content"
